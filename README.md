## Development

### Prerequisites

- Ruby >= 3.0
- Bundler gem
- Git

### Local Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/lbds137/the-night-house.git
   cd the-night-house
   ```

2. Install dependencies:
   ```bash
   bundle install
   ```

3. Run the site locally:
   ```bash
   bundle exec jekyll serve
   ```

4. Visit `http://localhost:4000` in your browser
